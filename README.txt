This is a social media website that I made to learn SignalR, Javascript and AJAX.

Features include: Real time messaging, uploading statuses (newsfeed automatically gets new statuses so no need to refresh),
			adding friends, real time notifications for messages and friend requests.




You may need to change the connection string in the app and web config files to the name of your sql server.

Run “update-database” from the “Data Layer” project in the Package manager console.
